# tinyutil/utilities

[![NPM Version][npm-image]][npm-url]
[![Dependency Status][deps-image]][deps-url]

Utility functions for a small studio websites.

## Installation

```sh
npm install @asmallstudio/utilities
```

## Testing

To run the mocha tests:

```sh
npm run test
```
