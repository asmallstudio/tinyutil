# tinyutil/components

[React](https://reactjs.org/) component library for a small studio websites.

## Installation

```shell
$ npm install @asmallstudio/components
```

## Usage

```javascript
import { ComponentName } from "@asmallstudio/components";
```

## Component Listing

- AmbiLink
- DesignGuide
- Icon
- LayoutGuide
- ReactMarkdownLink
